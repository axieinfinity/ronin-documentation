# Minimum hardware requirements

Running a node requires a lot of disk space and drives with fast read/write speed (IOPS)

## Running a full Ronin node

* 6 CPU cores
* 25 GB RAM
* 700 GB high-speed SSD

## Running an archive Ronin node

* 8 CPU cores
* 32 GB RAM
* 5 TB high-speed SSD
